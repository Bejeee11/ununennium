# Citation

If you use Ununennium in your research or publications, please cite it using the information below.

---

## Preferred Citation

### BibTeX

```bibtex
@software{ununennium2025,
  title = {Ununennium: Production-grade Satellite Imagery Machine Learning},
  author = {Laitinen Imanov, Olaf Yunus and Rzazade, Hafiz and Mamedova, Laman and Mirzaliyev, Farid and Ajili, Ayan},
  year = {2025},
  version = {1.0.5},
  url = {https://github.com/olaflaitinen/ununennium},
  note = {Production-grade Python library for Earth observation machine learning}
}
```

### APA Style

Laitinen Imanov, O. Y., Rzazade, H., Mamedova, L., Mirzaliyev, F., & Ajili, A. (2025). *Ununennium: Production-grade Satellite Imagery Machine Learning* (Version 1.0.5) [Computer software]. https://github.com/olaflaitinen/ununennium

### Chicago Style

Laitinen Imanov, Olaf Yunus, Hafiz Rzazade, Laman Mamedova, Farid Mirzaliyev, and Ayan Ajili. 2025. "Ununennium: Production-grade Satellite Imagery Machine Learning." Version 1.0.5. https://github.com/olaflaitinen/ununennium.
